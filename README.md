# Project Installation

```bash
   git clone https://github.com/bhattaraijay05/usp-project.git
   cd usp-project
```

# Setup

```bash
Install node js :
https://nodejs.org/en/download/
```

# Initialization of Backend

Create a virtual environment

```bash
cd backend
npm install -g yarn
yarn install
yarn start
```
